¿Que son? los componentes pasivos en la Red

Un componente pasivo, o dispositivo pasivo, es un componente electrónico que solo puede recibir energía, que puede disipar, absorber o almacenar en un campo eléctrico o magnético. Los componentes pasivos no necesitan ningún tipo de energía eléctrica independiente para funcionar.

¿Que son?

los componentes activos en la Red

¿Qué son los dispositivos Activos? Son aquellos dispositivos o equipos que se encargan de distribuir en forma activa la información a través de la red, como concentradores, access point, switches, router, entre otros. Además, se encargan de distribuir banda ancha a determinada cantidad de equipos en una red

Descripcion tecnica de cada componente Activo

Switch

Un switch es un dispositivo de red que conecta múltiples dispositivos dentro de una misma red local (LAN). Funciona en la capa de enlace de datos (Capa 2) del modelo OSI y permite la comunicación entre dispositivos mediante la conmutación de paquetes. Los switches pueden ser no gestionados, que son plug-and-play, o gestionados, que permiten configuraciones avanzadas como VLANs, QoS y monitoreo de tráfico. Los switches más comunes son los de 10/100/1000 Mbps (Gigabit Ethernet) y pueden tener desde 5 hasta más de 48 puertos.

Router

Un router es un dispositivo que dirige el tráfico de datos entre diferentes redes, como una red local y la Internet. Opera en la capa de red (Capa 3) del modelo OSI y utiliza direcciones IP para determinar el mejor camino para enviar los datos. Los routers pueden ofrecer funciones adicionales como cortafuegos, NAT (traducción de direcciones de red) y DHCP (asignación dinámica de direcciones IP). Existen routers de diferentes tipos, incluyendo routers de banda ancha, routers inalámbricos y routers de nivel empresarial.

Tarjeta de Red

Una tarjeta de red (NIC, por sus siglas en inglés) es un componente de hardware que permite a un dispositivo comunicarse en una red. Puede ser una tarjeta de expansión interna (PCI, PCIe) o un adaptador USB. Las tarjetas de red pueden ser de tipo Ethernet (cableadas) o inalámbricas (WiFi). Las especificaciones incluyen la velocidad de transmisión (10/100/1000 Mbps para Ethernet, o estándares como 802.11ac para WiFi) y la compatibilidad con diferentes protocolos de red. Interruptor Inteligente WiFi Un interruptor inteligente WiFi es un dispositivo que permite controlar dispositivos eléctricos de forma remota a través de una aplicación móvil o comandos de voz. Se conecta a la red WiFi del hogar y puede ser integrado con sistemas de automatización del hogar. Las características técnicas incluyen:

  • Compatibilidad con estándares como IEEE 802.11 b/g/n.
  • Capacidad para manejar cargas eléctricas de hasta 10A a 220V.
  • Funciones de programación y temporización.
  • Integración con asistentes de voz como Amazon Alexa y Google Assistant.

Access Point

Un access point (AP) es un dispositivo que permite la conexión de dispositivos inalámbricos a una red cableada. Funciona como un puente entre la red cableada y los dispositivos WiFi, extendiendo la cobertura de la red. Los access points pueden ser independientes o parte de un sistema de gestión centralizada. Las especificaciones incluyen:

  • Soporte para estándares WiFi como 802.11n, 802.11ac o 802.11ax (WiFi 6).
  • Capacidad de manejar múltiples conexiones simultáneas.
  • Funciones de seguridad como WPA3.
  • Posibilidad de ser alimentados por PoE (Power over Ethernet) para facilitar la instalación.

Fibra Óptica

La fibra óptica es un medio de transmisión que utiliza luz para enviar datos a través de filamentos de vidrio o plástico. Este tipo de cable tiene ventajas significativas sobre el cableado de cobre, como mayor ancho de banda, menor atenuación y resistencia a interferencias electromagnéticas. Existen dos tipos principales de fibra óptica: monomodo y multimodo, cada una adecuada para diferentes aplicaciones y distancias de transmisión.

Canaletas

Las canaletas son estructuras utilizadas para organizar y proteger el cableado en instalaciones de redes. Permiten una gestión ordenada de los cables, minimizando el riesgo de daños y facilitando el acceso a los mismos para mantenimiento. Las canaletas pueden ser de diferentes materiales, como PVC o metal, y están disponibles en varias dimensiones para adaptarse a diferentes necesidades de instalación.

Descripcion tecnica de cada componente pasivo

RJ45

El conector RJ45 es un tipo de conector de red utilizado principalmente para la conexión de dispositivos en redes Ethernet. Este conector tiene ocho pines y se utiliza comúnmente con cables de par trenzado, como el Cat5e, Cat6 y Cat6a. Los estándares de cableado más utilizados son T568A y T568B, que definen la disposición de los cables dentro del conector.

Jacks o Keystone

Los jacks RJ45 tipo keystone son módulos modulares diseñados para ser instalados en paneles de conexión o placas de pared. Estos jacks cumplen con normas como IEC 60603-7 y TIA 568-C.2, y son compatibles con PoE+ (Power over Ethernet). Tienen características como:

  • Resistencia de aislamiento mínima de 500Ω.
  • Rigidez dieléctrica de 1000 VCA.
  • Corriente máxima de 1.5A por conductor a 20°C.
  • Disponibles en varias categorías, incluyendo Cat5e, Cat6 y Cat6a, con recubrimiento de 50µm de oro sobre níquel en los contactos.

Cable UTP

El cable UTP (Unshielded Twisted Pair) es un tipo de cable de par trenzado que no tiene blindaje. Se utiliza en aplicaciones de red para transmitir datos. Existen diferentes categorías de cables UTP, como Cat5e, Cat6 y Cat6a, que varían en capacidad de transmisión y velocidad. Por ejemplo, el cable Cat6 puede soportar velocidades de hasta 10 Gbps a distancias de hasta 55 metros.

Patch Panel

Un patch panel es un dispositivo que permite la gestión y organización de cables de red. Los paneles de conexión RJ45 pueden tener diferentes configuraciones, como paneles apantallados o no apantallados, y pueden variar en el número de puertos (12, 24, 48 puertos, etc.). Son esenciales para crear un sistema de cableado flexible y ordenado, facilitando la conexión de dispositivos en redes LAN.

Patch Cord

Un patch cord es un cable corto que se utiliza para conectar dispositivos a un patch panel o a un switch. Estos cables pueden ser de diferentes categorías, como Cat5e o Cat6, y están disponibles en varias longitudes y colores para facilitar la organización del cableado.

Herramientas utilizadas para realizar cableado estructurado.

Ponchadora

Una ponchadora, también conocida como crimpadora, es una herramienta utilizada para conectar terminales o conectores a cables de red, como RJ45, RJ11 y RJ12. Las características técnicas incluyen:

  • Diseño de acción compuesta que ejerce mayor potencia de ponchado con cada apriete.
  • Matraca incorporada que garantiza un ponchado de ciclo completo para un resultado uniforme.
  • Cavidades de ponchado codificadas por color para diferentes tamaños de terminales (ej. 22-16 AWG, 16-14 AWG, 12-10 AWG).
  • Perilla ajustable para controlar la fuerza de ponchado.
  • Mangos de doble material para mayor comodidad y agarre.
  • Capacidad para ponchar cables sólidos y trenzados de 10-22 AWG.

Desforradora

Una desforradora, también llamada pelacables, es una herramienta diseñada para quitar el aislamiento de cables de manera rápida y precisa. Características:

  • Hojas afiladas que permiten pelar el cable sin dañar los conductores.
  • Ajuste de profundidad para controlar la longitud del aislamiento retirado.
  • Diseño ergonómico para un uso cómodo y eficiente.

Tester/Probador de cables

Un tester o probador de cables es un dispositivo que se utiliza para verificar la integridad y el cableado de cables de red. Puede detectar fallos como cables abiertos, cortocircuitos y pares cruzados. Características:

  • Pantalla LCD que muestra el estado del cable y los pares.
  • Funciones de prueba para cables UTP, STP y coaxiales.
  • Capacidad para probar cables Ethernet, telefónicos y de audio.
  • Alimentación por batería para pruebas portátiles.
